# OpenDCI

Open DOCSIS Configuration Interchange format

## Reference Specifications

OpenDCI is based on the following CableLabs specifications. See [`docs/SPECS.md`](docs/SPECS.md) for download instructions and local copies in `docs/external/`.

| Specification | Description |
|---------------|-------------|
| [CM-SP-MULPIv3.1](https://www.cablelabs.com/specifications/CM-SP-MULPIv3.1) | DOCSIS 3.1 MAC and Upper Layer Protocols -- definitive source for configuration file structure |
| [CM-SP-MULPIv3.0](https://www.cablelabs.com/specifications/CM-SP-MULPIv3.0) | DOCSIS 3.0 MAC and Upper Layer Protocols |
| [CL-SP-CANN](https://www.cablelabs.com/specifications/CL-SP-CANN) | Assigned Names and Numbers -- central registry for all TLV values and sub-TLVs |
| [CM-SP-CM-OSSIv3.1](https://www.cablelabs.com/specifications/CM-SP-CM-OSSIv3.1) | DOCSIS 3.1 CM Operations Support System Interface -- config file management and monitoring |
| [CM-SP-OSSIv3.0](https://www.cablelabs.com/wp-content/uploads/2015/08/CM-SP-OSSIv3.0-I05-071206.pdf) | DOCSIS 3.0 OSSI (freely downloadable) |
